# UI Rules

## Mobile First
- Prioridade total para celular
- Layout limpo
- Alta legibilidade
- Sem excesso de altura

## Tipografia
- Forte
- Clara
- Escaneável
- Sem letter spacing exagerado

## Espaçamento
- Premium
- Consistente
- Respiro visual
- Sem blocos gigantes

## Visual
- Aparência moderna
- Sensação premium
- Alta percepção de valor
- Contraste forte
- Hierarquia visual clara

## Conversão
- CTA muito visível
- Hero claro
- Benefícios rápidos de entender

## Evitar
- Visual genérico
- Cards ruins
- Muito texto
- Efeitos exagerados
- UI robótica
